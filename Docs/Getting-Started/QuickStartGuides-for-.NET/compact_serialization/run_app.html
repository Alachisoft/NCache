Step 3: Run Your Application

Before running your application, make it sure you have copied the assembly containing compact types (if it is using Read-Through/Write-Through) in the 'Service' folder of NCache installation directory (%INSTALL_DIR%/NCache/bin/service). 

Also, make sure compact types are  registered with the cache and cache is running if it is configured as OutProc cache. You may see the statistics of the cache through statistics window.
