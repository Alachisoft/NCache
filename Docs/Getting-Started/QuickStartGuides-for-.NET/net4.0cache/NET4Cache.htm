Quick Start with .NET 4.0 Cache Provider

Following steps need to be followed for using cache provides in your application:


  1. Create an application inside Visual Studio.NET.
  2. Add following references to your application from "Installed Directory/NCache/integration/DotNet4.0 Cache Provider" path:

    3. "Alachisoft.NCache.ObjectCacheProvider"


  3. Include the following namespace in your project.


    4. using Alachisoft.NCache.ObjectCacheProvider;


  4. Initialize your CacheProvider and pass your cache name to the provider as shown below:


    5. private ObjectCache _cache;

    private string _cacheId = "myCache" ;

    _cache = new CacheProvider(_cacheId);


  5. Now you can perform all cache related operations on your cache using CacheProvider commands.
