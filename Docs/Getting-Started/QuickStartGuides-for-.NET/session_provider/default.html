Using Session State Store Provider

NCache Session State Provider is a custom SessionStateStoreProviderBase implementation for an ASP.NET application. This implementation reads and writes ASP.NET session data from and to the cache. This allows you to transparently save and retrieve ASP.NET session-state objects within a webfarm NCache also provides the Session State Module which can be used for applications that are intended to run on older .NET frameworks like 1.1.
