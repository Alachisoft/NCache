Step 3: Start/Restart ASP.NET Application

Now that you have configured a cache through NCache Manager and have edited web.config to add NCache Output Caching, you're ready to start or restart your ASP.NET application. To restart your ASP.NET application, kill your ASP.NET process through Task Manager first.
