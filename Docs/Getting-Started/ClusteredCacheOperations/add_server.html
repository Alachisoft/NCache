Add Server Node

A Server Node can be added in a cluster any time. After adding a server node in a cluster, you are supposed to start the node so that it can join the cluster. To add a node to a cluster, right click on cluster in the 'NCache Management' tree and select 'Add Server Node' from the context menu.



In either case, a wizard will appear with the list of available nodes. Select a node from the list and click 'Finish'. 



The new node will become part of the cluster with the same configurations as of the cluster. A warning message will be shown in case of a port conflict.
