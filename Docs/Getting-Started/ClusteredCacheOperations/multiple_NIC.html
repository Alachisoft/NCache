Select Network Interface Card

If your cache servers (including your app servers if you're hosting cache on them) have multiple network cards, then you must bind NCache to a specific card. You can also bind Client/Server communication to different network cards. Or, you can bind them both to the same card. NCache can detect all Network Interface Cards (NIC's) on a server node and allows you to select among them. Selected NIC will be used by NCache for cluster-wide communication.

Following steps are required to select NIC:


  1. Right click on a server node and select 'Select Network Interface Card' option from the context menu.


    NCache will detect and list all the NIC's on that node. Current NIC will be checked in green.


  2. Select NIC of your own choice and press 'Apply'.



  3. Click 'Apply'. You will be prompted to restart NCache Service to work with fresh settings.



  4. Click 'Yes'.

Manual modification of NIC's IP


The selected NIC’s IP address is written in the appSettings of Alachisoft.NCache.Service.exe.config file. You can also manually modify the NIC's IP through configuration file, as shown below:


  1. Go to Alachisoft.NCache.Service.exe.config located at:
    %INSTALLATION_PATH%/NCache/bin/service/Alachisoft.NCache.Service.exe.config
  2. Add following line in the configuration file:

    <add key = "NCacheServer.BindToClusterIP" value = "" />

    <add key = "NCacheServer.BindToClientServerIP" value = "" />


    BindToClusterIP: You can inform NCache which interface to use for cluster-wide communication. In order to do so, provide the Ip Address you want the cluster server to bind with.

    BindToClientServerIP: You can inform NCache which interface to use for client communication. In order to do so, please provide the Ip Address you want the client's (NCache client) server to bind with.

In the case your cache cluster consists of two or more nodes, the client.ncconf of other nodes is not updated. You will have to use "Configure Remote Clients" to update client.ncconf of other nodes.
