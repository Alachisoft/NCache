Start/Stop a Cache Cluster

To start a cluster, right click on the cluster in the 'NCache Management' tree and select 'Start' from the context menu. It will start all the server nodes in a cluster.



To stop a cluster, right click on the cluster in the 'NCache Management' tree and select 'Stop' from the context menu.
