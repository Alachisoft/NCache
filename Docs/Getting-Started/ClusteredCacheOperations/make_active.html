Make a node Active

In Mirror Cache you have two nodes, Active and Passive. The default node will act as an Active node, where all clients are connected. You can make a passive node active any time. To make a node active, right click on the passive node and select 'Make Active Node' option from the context menu. If the Mirror Cache is stopped then status of the nodes is immediately changed. The passive node becomes active and the active node becomes passive. But if Mirror Cache is running then the status of the nodes is changed only after the cache is restarted.
