Add Existing Cluster

To add an existing Cluster, right click on 'Clusters' from the 'NCache Management' tree and select 'Add Existing Clustered Cache' option from the context menu.



An 'Add Clustered Cache Wizard' will appear with a list of available Clusters. Steps to add an existing cluster through wizard are as follows:


Step 1 of 2


Select a node: Select a running node from where you want to get the existing cluster. You can specify an IP address or select one from the list of available nodes.



Step 2 of 2


Select a clustered cache: Select a clustered cache from the list of available clustered caches on this node. Click finish to end the wizard.
