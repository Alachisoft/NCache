Restart a Cluster

Before restarting a cluster, make it sure that your application is not in use otherwise it will throw an exception. "Restart" option will remove all data from the cache. To restart a cluster, right click on the cluster in the 'NCache Management' tree and select 'Restart' from the context menu. It will stop all the server nodes in a cluster and give a fresh start to all server nodes.
