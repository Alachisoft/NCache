Start/Stop a Cache Server

To start a server node, right click on the node in the 'NCache Management' tree and select 'Start' from the context menu. On starting a new server node in a cluster will replicate the already existing data on that node (in case of replicated topology).



To stop a server node, right click on the node  in the 'NCache Management' tree and select 'Stop' from the context menu.
