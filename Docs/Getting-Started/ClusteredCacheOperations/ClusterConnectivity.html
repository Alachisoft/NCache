Cluster Connectivity

Connectivity among the nodes of a cluster is very important for cluster operations. Therefore checking the connectivity of cluster is necessary. To check the connectivity of a cluster, NCache provides a dialog showing connectivity among the nodes of specified cluster and details about the status of each node in the cluster.

To View cluster connectivity of two nodes cluster, right click on cluster in the 'NCache Explorer' tree and select 'View Cluster Connectivity' from the context menu.



The dialog given below will appear, showing all the details about connectivity of each node in that cluster.
