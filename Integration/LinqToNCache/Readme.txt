NCACHE LINQ
-----------
This dll contains the provider implemented to give user the ability to use LINQ queries with NCache.

To use this feature, reference the provided dll, in your application.

For more information, refer to NCache Help->Developing with NCache->Programmers Guide->Using Queries->Using LINQ.

NOTE
----
For sample application goto samples(in NCache folder)->clr20->NCacheLINQ.
